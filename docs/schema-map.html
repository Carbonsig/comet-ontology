The COMET Ontology, Mapped

Every class, property, individual and standards alignment in one place — explore it as an interactive graph, read it as a layered schema, sort and search the full value lookup, or download the complete spreadsheet.

Drag nodes · scroll to zoom · click a node to inspect and isolate its neighbourhood · click empty space to clear · toggle namespaces in the legend. The graph shows the relational COMET ontology (7 core layers + 2 extensions + standards alignments); the 649 flat CAD Trust dictionary fields live in the Value Lookup tab and the XLSX.

Layers shown in greyed type (Core, Emission Factor, Supply Chain, Market, Verification) are referenced by the extension modules but their base vocabularies ship in the COMET core distribution rather than this repository — counts reflect terms defined here.
